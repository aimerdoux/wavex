# WaveX NFT Project

## Setup
1. Clone repository
2. Install dependencies: `npm install`
3. Create `.env` file with `ALCHEMY_API_KEY` and `PRIVATE_KEY`

## Compile Contract
`solcjs --bin --abi WaveXNFT.sol -o build`

## Deploy Contract
`node deploy.js`

- GitHub repository for storage
